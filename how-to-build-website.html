Want to Make Your Own Website?

I built Cool Games 64x all by myself! You can too. This guide will show you how I did it, step by step. No computer genius required - if I can do it, YOU can do it! 🚀

1

What Do You Need?

Before we start, here's what you'll need:

🛠️ Tools You'll Need:

  • A computer - Any laptop or desktop will work
  • An internet connection - To save your website online
  • A web browser - Like Chrome, Safari, or Edge
  • Github account - FREE! This is where your website lives
2

Create Your Github Account

GitHub is like a magical cloud storage for your website! It's where programmers store their projects. For kids, it's the easiest way to put a website online for FREE.

💡 How to sign up:

1. Go to github.com

2. Click "Sign Up"

3. Choose a username (pick something cool!)

4. Enter your email and a password

5. Verify your email -非常重要!

3

Create Your First Repository

A "repository" (or "repo") is just a folder where your website files live. Think of it like a folder on your computer, but it lives on the internet!

💡 How to create a repo:

1. Click the + button (top right)

2. Click "New repository"

3. Name it like this:

// Your repo name should be exactly like this:
yourusername.github.io

(Replace "yourusername" with YOUR actual GitHub username!)

4. Make it PUBLIC

5. Click "Create repository"

4

Write Your First HTML File

HTML is the language that makes websites work! Every webpage is written in HTML. It looks complicated but it's actually super simple!

<!-- This is how you start every HTML file -->
<!DOCTYPE html>
<html>
<head>
  <title>My First Website!</title>
</head>
<body>
  <h1>Hello World!🌎</h1>
  <p>This is MY website!</p>
</body>
</html>

Copy this into a text editor (like Notepad on Windows or TextEdit on Mac), save it as index.html, and that's your first webpage!

5

Upload to GitHub

Now we need to put your file on GitHub so the whole world can see it!

💡 How to upload:

1. Go to your repo on github.com

2. Click "Add file" → "Upload files"

3. Drag your index.html file there

4. Scroll down and click "Commit changes"

5. Go to Settings → Pages (on the left)

6. Under "Source", click "Deploy from a branch"

7. Select "main" and "/ (root)"

8. Click Save!

6

Wait and Celebrate! 🎉

GitHub will take a few minutes to publish your site. Once you see a green checkmark, your website is LIVE!

🎊 YOUR WEBSITE IS NOW LIVE AT:

https://yourusername.github.io/

Share this link with your friends, family, and everyone! They can now see your website on ANY device - phone, tablet, computer - it works everywhere!

🌟

Cool Things to Add Next

Now that your basic site is working, here are some fun things to learn next:

🚀 Level Up Your Website:

  • CSS - Add colors, fonts, and cool effects!
  • JavaScript - Make things move and click!
  • Games - Add your own playable games!
  • Blog - Write about what you learn!

⚡ Remember!

Everyone starts as a beginner! I made lots of mistakes when I was learning, and that's TOTALLY normal. The important thing is to keep trying and not give up! Every expert was once a beginner. 💪
